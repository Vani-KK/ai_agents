# Research Summary Assistant v2
An upgraded AI research assistant with multi-source synthesis and a Streamlit web interface.

## Features
- Searches web using DuckDuckGo

- Reads and extracts 3 articles simultaneously

- Synthesizes multiple sources into structured report

- Streamlit web UI with download option

- Uses LangChain and GPT-4o-mini



## Installation

Clone the repository:

git clone https://github.com/Vani-KK/ai_agents.git

cd ai_agents/research_summary_assistant


## Environment Setup

Create a .env file in the project folder and add your API key:

OPENAI_API_KEY=your_api_key_here

- .env is ignored using .gitignore so that API keys are not uploaded to GitHub.


## Technologies Used:

 - Python
 - Requests library
 - OpenAI API
 - BeautifulSoup
 - Streamlit
